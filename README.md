📊 Customer Churn Analysis Dashboard

This project presents a Customer Churn Analysis Dashboard built using Power BI to identify churn patterns, retention trends, and key factors influencing customer loss in a subscription-based business.

🔍 Project Objective

The main goal of this project is to:

Analyze customer behavior and churn trends

Identify high-risk customers

Discover key retention drivers

Provide actionable business insights to reduce churn

🛠 Tools Used

Power BI

Data Cleaning & Transformation

Data Visualization

KPI & Retention Analysis

📈 Key Insights

Churn rate analysis by customer segment

Revenue impact due to churn

Customer lifetime value trends

Retention comparison across different categories

📌 Features of the Dashboard

Interactive filters and slicers

KPI cards for quick performance overview

Visual comparison of churn vs retained customers

Trend analysis charts

🚀 Outcome

This dashboard helps businesses make data-driven decisions to improve customer retention strategies and reduce revenue loss.
